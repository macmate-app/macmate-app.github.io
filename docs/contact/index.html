How to

In order for us to better service you, please do include the following information in your email:

  1. App name and version
  2. Operating system and version
  3. Steps to reproduce the issues/bugs if there are any
  4. A screenshot is very helpful for us to quickly know what is happening
  5. Any other information which might help us to troubleshot the issue
